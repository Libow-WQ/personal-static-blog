# 发布成项目链接

## 方案一：Netlify Drop

1. 打开 https://app.netlify.com/drop
2. 将整个 `static-blog` 文件夹拖进去
3. 等待发布完成
4. 复制生成的 `https://xxx.netlify.app` 链接放到简历

## 方案二：GitHub Pages

1. 新建 GitHub 仓库，例如 `personal-static-blog`
2. 上传 `index.html`、`articles.html`、`styles.css`
3. 进入仓库 `Settings` -> `Pages`
4. Source 选择 `Deploy from a branch`
5. Branch 选择 `main`，目录选择 `/root`
6. 保存后等待生成 `https://你的用户名.github.io/personal-static-blog/`

## 推荐简历链接写法

项目预览：填写发布后生成的 Netlify 或 GitHub Pages 链接

项目源码：填写 GitHub 仓库链接
