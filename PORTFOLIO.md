个人博客静态页面

## 项目链接

在线预览：https://libow-wq.github.io/personal-static-blog/

源码地址：https://github.com/Libow-WQ/personal-static-blog

## 项目简介

该项目是一个使用 HTML5 + CSS3 独立完成的个人技术博客静态页面，包含首页和文章列表页。页面采用 Flexbox + Grid 混合布局，实现桌面端、平板端、手机端响应式适配，适合作为前端基础能力和页面布局能力展示作品。

## 技术栈

HTML5、CSS3、Flexbox、Grid、CSS Variables、GitHub Pages

## 个人负责内容

- 完成首页和文章列表页的页面结构设计与编码
- 使用 CSS Grid 搭建页面主布局
- 使用 Flexbox 完成导航栏、卡片内容和按钮区域排版
- 使用 CSS Variables 统一管理主题色
- 使用纯 CSS `:checked` 实现移动端汉堡菜单展开和收起
- 完成三端响应式适配和 GitHub Pages 部署

## 项目亮点

- 无第三方库依赖，代码结构清晰，适合快速加载
- 使用 `:root` 变量集中管理主题色，便于换肤和后期维护
- 移动端菜单不依赖 JavaScript，使用纯 CSS 完成交互
- 页面内容完整，包含个人介绍、文章卡片、分类、归档和页脚信息
- 已部署为可访问的线上项目链接，可直接放入简历或作品集
