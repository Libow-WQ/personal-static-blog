# 个人博客静态页面

这是一个基于 HTML5 + CSS3 开发的个人技术博客静态页面项目，包含首页和文章列表页，适合作为前端基础项目或简历作品展示。

## 项目亮点

- 独立完成 1 个首页 + 1 个文章列表页
- 使用 Flexbox + Grid 混合布局，实现桌面端、平板、手机端响应式适配
- 使用 CSS 自定义属性 `:root` 管理主题色，主色为 `#2D7CF6`
- 支持纯 CSS 一键换肤能力，无需 JavaScript
- 移动端导航栏折叠为汉堡菜单，使用 `:checked` 伪类控制展开和收起
- 无第三方库依赖，页面结构轻量，适合快速加载

## 文件结构

```text
static-blog/
├── index.html       # 首页
├── articles.html    # 文章列表页
└── styles.css       # 共用样式
```

## 本地运行

直接双击打开 `index.html` 即可预览。
